System Requirements

  • Windows 10 (1703 or later) or Windows 11
  • Microsoft Edge WebView2 Runtime (the installer adds this automatically when missing; included with Windows 11 and current Windows 10)
  • An email account. QuickMail supports Microsoft 365 / Exchange Online and Outlook.com (through Microsoft 365 directly), Gmail, iCloud, and any standard IMAP/SMTP provider. Work or school Microsoft 365 accounts may first need approval from an organization administrator — see For Exchange / Microsoft 365 Administrators.
